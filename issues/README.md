# Issues Planning

This folder contains planning drafts for GitHub feature issues and sub-issues.

Approach:

- One file per feature for better issue authoring and sub-issue decomposition.
- Milestone-first organization so epic-to-feature relationships are explicit.

Current scope:

- Milestone 01 drafts are in `issues/milestone-01-foundation-and-azure-skeleton`.
- Milestone 02 drafts are in `issues/milestone-02-auth-and-invite-only-access`.
- Milestone 03 drafts are in `issues/milestone-03-cosmos-db-persistence`.

Navigation:

- Each milestone folder has an `index.md` file listing its feature issue drafts.
